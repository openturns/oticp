# oticp module

Examples
  Template examples can be found here: C:\Python27\Lib\site-packages\oticp\examples\oticp

Installation
  Silent install can be done with the switch /S (e.g. openturns-oticp-0.1-py27.exe /S).
  To test oticp module installation, in an OpenTURNS python console type:
    import oticp
    print oticp.__version__

Uninstallation
  OTICP module will be automatically uninstalled at OpenTURNS removal.
  OTICP module can be removed from OpenTURNS with uninst-template.exe uninstaller.
  Silent uninstall can be done with the switch /S.
